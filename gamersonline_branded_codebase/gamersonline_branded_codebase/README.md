# GamersOnline.gg

Branded monorepo starter for a competitive CS2 platform.

## Included
- GamersOnline red / blue / white landing page
- Next.js frontend with login, signup, dashboard, queue, and billing screens
- NestJS API with auth, billing, queue, launcher, anti-cheat, and admin case shells
- Worker scaffold for queue processing
- Prisma schema for PostgreSQL
- Docker Compose for PostgreSQL + Redis
